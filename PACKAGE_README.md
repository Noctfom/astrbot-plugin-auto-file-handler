# AstrBot自动文件处理器插件开源包

## 📦 包含内容

### 核心文件
- `main.py` - 插件主程序 (v1.5.12)
- `plugin.json` - 插件元信息
- `_conf_schema.json` - 配置模式定义
- `README.md` - 详细使用说明

### 开源工具
- `github_push_guide.md` - GitHub推送完整指南
- `push_to_github.sh` - 自动化推送脚本

## 🚀 使用步骤

1. **部署插件**
   - 将核心文件复制到AstrBot的plugins目录
   - 重启AstrBot即可使用

2. **开源分享**
   - 阅读 `github_push_guide.md` 了解详细步骤
   - 修改 `push_to_github.sh` 中的配置变量
   - 运行脚本推送至GitHub
